# HBnB Evolution 🏡

HBnB Evolution is a simple backend application that replicates the basic functionalities of the Airbnb application.
Users can register, add places for rent, and leave reviews.

Currently, this repository contains the system architecture and UML diagrams.
These blueprints include the Package Diagram, Class Diagram, and Sequence Diagrams.

In the next phases, we will build the API and connect the database.
